# Spring Boot with Apache Kafka and MySQL (JdbcTemplate)
Simple project with Spring Boot, Apache Kafka and MySQL (JdbcTemplate). The purpose of this code is just to test of write and read performance of Apache Kafka messages.

## Requeriments
 - [MySQL](https://dev.mysql.com/downloads/)
 - [Apache Zookeeper](https://zookeeper.apache.org/releases.html)
 - [Apache Kafka](https://kafka.apache.org/downloads)
